A simple interactive command-line game built with **Dart** to demonstrate the implementation of fundamental programming concepts including variables, data types, loops, and branching logic.

## Features
- **Dynamic Input**: Accepts real-time user input via the terminal using `dart:io`.
- **Game Logic**: Implements a "Secret Number" system with a limited life/chance mechanism.
- **Branching**: Uses `if-else` statements to provide feedback (Too High/Too Low).
- **Looping**: Utilizes `while` loops for continuous gameplay until a win/loss condition is met.
- **History Tracking**: Uses a `List` (Non-primitive data type) to store and display all user attempts at the end of the game.
